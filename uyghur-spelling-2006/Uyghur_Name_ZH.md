# 关于《维吾尔》一词英文转写的意见

近年来随着改革开放的不断深化，各行各业人员与外界的经济贸易往来和信息方面的互通日益频繁，再加上新闻出版和外事活动方面的往来增多，对新疆的主体民族——维吾尔族名称英文转写的统一和规范提出了新的要求。

目前，《维吾尔》一词在社会上的英文转写和使用情况非常混乱。经调研发现《维吾尔》一词在社会上的英文转写共有 Uyghur, Uygur, Uighur, Uighuir, Uiguir, Uigur 和 Weiwuer 等多种。这种情况给人们的工作和生活都带来了诸多不便。鉴于上述情况，新疆维吾尔自治区民族语言名词术语规范审定委员会经研究并征求有关专家意见后，将《维吾尔》一词的英文转写方式和读音规范为 Uyghur 和 [ujɣur]。

望有关部门和个人遵照使用。

新疆维吾尔自治区民族语言名词术语规范审定委员会

2006年10月11日
